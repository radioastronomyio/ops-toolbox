# Ops Toolbox Memory Bank

This directory contains the persistent context for AI agents working on the ops-toolbox project.

## Files

| File | Purpose | Update Frequency |
|------|---------|------------------|
| **brief.md** | Foundational 2-3 paragraph context | Rarely |
| **product.md** | Why this exists, problems solved, goals | Occasionally |
| **context.md** | Current state, next steps, active decisions | **Every session** |
| **architecture.md** | System design, structure, patterns | When patterns change |
| **tech.md** | Technologies, setup, constraints | When stack changes |
| **tasks.md** | Repetitive workflows and procedures | As workflows emerge |

## Quick Start for Agents

1. Read **brief.md** for foundational context
2. Read **context.md** for current state and next steps
3. Reference **architecture.md** and **tech.md** as needed during work
4. **Update context.md** at session end
